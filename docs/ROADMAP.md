# SkitBox Roadmap

## Current Patch Line

The `v0.3.x` line is for first-run friction only:

- download ZIP
- unzip
- double-click
- generate a skit
- run Tester
- save canon
- export a share card
- leave feedback

## v0.4 Candidate

The next feature pass should be fun and share focused:

- better share-card layouts for best lines
- a few bundled World Pack examples
- clearer world import/export wording
- a small gallery of reproducible funny seeds
- optional screenshot-friendly output mode

Avoid adding AI providers, accounts, npm, or cloud services in this pass.

## Packaging Decision

Stay with the ZIP plus Python launcher until tester feedback proves Python is the main blocker.

Move toward an EXE release candidate only if people repeatedly fail before the browser opens.
